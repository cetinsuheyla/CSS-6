This page is all about toggling effect.

How do you do that?

  1. Use input and give 'checkbox' for type attribute.
  2. Create label for this checkbox.
  3. In nav element (after this input) add your ul and li elements, put anchor tag in each li element.
  4. Add display: none for input, this way you can only see the label.
  5. Make nav element's height 0, and overflow hidden in order not to see it. You will need to see it only when input is checked.
  6. When input is checked give nav element's max-height 500px.
  7. It works! If you want to make additional styling add colors with hover effect and add borders for list elements accept the last list element.
